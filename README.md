## Watermark Images

### About
This project adds watermark text and logo to the bottom right of the image to images uploaded from the computer and saves them into the directory under sample-out.png. It uses the Pillow library, which is a fork of the PIL (Python Imaging Library) library and Tkinter for the GUI.
